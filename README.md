# Python projects
Python projects from Programming Fundamentals coursework from ACC

Projects:
- Hot Dog Cookout Calculator – calculates the number of hot dog and bun packages needed for a cookout.
- Character Analysis Program – reads a text file and counts uppercase letters, lowercase letters, digits, and whitespace.
- Pennies to Pay Program – calculates total pay over a period of days with increasing daily earnings.

These projects demonstrate programming concepts such as loops, conditionals, file handling, input validation, and formatted output.
